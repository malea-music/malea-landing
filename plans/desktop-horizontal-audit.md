# Desktop Horizontal Audit — MALEA «Сакральный артист»

> Аудит desktop-версии (hover:fine pointer:fine) для viewport'ов 1366×768, 1440×900, 1536×864, 1920×1080.
> Никакие файлы не изменялись. Только анализ.

---

## Базовые параметры аудита

| Параметр | Значение |
|---|---|
| Media query | `@media (min-width: 961px) and (hover: hover) and (pointer: fine)` |
| Доп. desktop MQ | `@media (min-width: 1200px)` — увеличенный gap для philosophy / live-text |
| CSS variables desktop | `--f-hero: clamp(72px, 8.4vw, 104px)`; `--f-section: clamp(44px, 4.8vw, 76px)` |
| Layout-паттерн | grid `52%/48%` или `46%/54%` split; текстовые секции padding `var(--col-pad)` |

## Топ-5 приоритетных desktop-проблем

| # | Экран | Проблема | Приоритет | Тип |
|---|---|---|---|---|
| 1 | 02 · hero | Кнопка может быть ниже viewport на 1366×768 | High | composition |
| 2 | 26 · portfolio-quote | `100vh` + `translateY(-2vh)` — риск артефактов | Medium | vertical height |
| 3 | 08 · player | Свободное пространство снизу на 1366×768 | Low | vertical height |
| 4 | 25 · portfolio-map | Длинные названия событий — плотная ширина на 1366 | Low | typography |
| 5 | 12 · musicians-gallery | Пропорции фото на 1366×768 | Low | image crop |

---

## Рекомендованный порядок исправлений

1. **Hero (02)** — самая заметная проблема (CTA может быть не видна без скролла на 1366×768). Проверить `min-height` или padding.
2. **Portfolio-quote (26)** — `100vh` + `translateY` может давать смещение. Рассмотреть `min-height: 100dvh` вместо `height: 100vh`.
3. **Player (08)** — опционально: уменьшить padding-top/bottom или установить `height: auto` для 1366.
4. **Portfolio-map (25)** — проверить `font-size` названий событий на 1366×768.
5. **Musicians-gallery (12)** — проверить `aspect-ratio` фото.
